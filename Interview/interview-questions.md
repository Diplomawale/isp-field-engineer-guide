# Telecom & Networking Interview Questions 🎯

This section contains important questions for:
- ISP Field Engineer
- Network Engineer (Beginner Level)

---

## 📌 Basic Questions

### 1. What is UBR?
UBR stands for Uncommitted Bit Rate. It is used when bandwidth is not guaranteed and depends on network availability.

---

### 2. What is LOS?
LOS means Line of Sight. It ensures a clear path between two antennas for wireless communication.

---

### 3. What is IP Address?
An IP address is a unique address used to identify a device in a network.

---

### 4. What is Gateway?
Gateway is the device that connects one network to another.

---

### 5. What is Ping?
Ping is used to check connectivity between two devices.

---

## 🔧 Practical Questions

### 6. What will you do if link is down?

- Check power supply
- Check cable connection
- Check IP configuration
- Perform ping test
- Verify LOS

---

### 7. How do you check LOS in field?

- Check obstacles
- Use Google Earth
- Adjust antenna height
- Align antenna direction

---

### 8. What is troubleshooting?

Troubleshooting is the process of identifying and solving network problems.

---

## 📡 Field Engineer Questions

### 9. What tools do you use in field?

- Google Earth
- AirLink
- GPS
- Compass

---

### 10. How do you configure a device?

- Assign IP address
- Set gateway
- Configure interface
- Test connectivity

---
