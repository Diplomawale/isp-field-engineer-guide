## 📌 What is UBR?

UBR has **two meanings** depending on context:

---

### 1️⃣ UBR (Uncommitted Bit Rate)

- A **bandwidth type**
- No guaranteed bandwidth
- Uses available network capacity
- Common in ISP networks

👉 Example: Shared internet connection

---

### 2️⃣ UBR (Unlicensed Band Radio) – Field Term

- Wireless communication using **license-free frequency**
- Used in **WISP / ISP networks**
- Supports P2P & P2MP links

---

## 📡 Frequency Bands Used in India

Wireless ISP networks use:

| Band | Frequency Range | Usage |
|------|---------------|------|
| 📶 2.4 GHz | 2400 – 2483.5 MHz | Short range |
| 📶 5 GHz | 5150 – 5875 MHz | Backhaul / Long range |

👉 These are **unlicensed bands** (no license required)

---

## 🔗 Network Types (VERY IMPORTANT)

### 🔹 Point-to-Point (P2P)

👉 One-to-One connection

Tower A  -------->  Tower B


- Dedicated link  
- High speed  
- Used for backhaul  

---

### 🔹 Point-to-Multipoint (P2MP)

👉 One-to-Many connection

Tower
   /  |  \
 User User User

 - Shared bandwidth  
- Cost-effective  
- Used for ISP distribution
