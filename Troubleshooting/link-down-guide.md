# Link Down Troubleshooting Guide 🔧

## 📌 Common Problem

Link Down means:
- No connectivity between devices
- No internet access
- Interface showing down

---

## 🚨 Possible Causes

- Power issue
- Cable problem
- Wrong IP configuration
- Interface shutdown
- LOS (Line of Sight) issue

---

## 🛠 Step-by-Step Troubleshooting

### 1️⃣ Check Power
- Ensure device is ON
- Check power adapter

---

### 2️⃣ Check Cable
- Verify LAN cable connected properly
- Replace cable if damaged

---

### 3️⃣ Check Interface Status
- Ensure port is UP
- Check LED indicators

---

### 4️⃣ Ping Test
- Ping gateway IP
- Ping remote device

---

### 5️⃣ Verify IP Configuration
- Check IP address
- Check subnet mask
- Check gateway

---

### 6️⃣ Check LOS (Wireless)
- Ensure no obstacle between antennas
- Adjust antenna direction

---

## 📡 Field Tips

- Always carry spare cable
- Keep backup power adapter
- Use mobile hotspot for testing

---

## ⚠️ Important

Always follow troubleshooting step by step.  
Do not skip steps — this avoids confusion.

---

## 🎯 Conclusion

Link down issues can be solved easily if:
- Proper steps are followed
- Basic checks are done correctly
